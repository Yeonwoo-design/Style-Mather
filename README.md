#Style-Mather
Style Matcher (디자인 조합 추천기)
💡 감성 키워드 기반 컬러 팔레트 & 폰트 조합 추천 웹 애플리케이션디자이너나 비전공자 누구나 쉽고 빠르게 '느낌(Vibe)'에 맞는 디자인 영감을 얻을 수 있도록 컬러 팔레트와 폰트 조합을 자동으로 제안하고 시각적으로 구성하는 웹 앱입니다.
📌 개요 : Style Matcher는 사용자가 원하는 디자인의 **'느낌(Feeling)'**을 키워드로 선택하면, 그 감성에 가장 잘 어울리는 **컬러 팔레트(Color Palette)**와 **폰트 조합(Font Combination)**을 실시간으로 제안하는 도구입니다. 복잡한 디자인 이론 없이도 직관적이고 시각적인 인터페이스를 통해 최적의 디자인 요소를 쉽게 찾고 적용할 수 있도록 돕습니다.
⚙️ 주요 기능 : 기능흐름설명감성 키워드 선택키워드 선택'모던', '빈티지', '활기찬' 등 다양한 감성을 대표하는 키워드를 선택합니다.자동 색상 팔레트 제안색상 선택선택된 키워드에 맞춰 주조색, 보조색, 강조색 등 3~5가지 색상으로 구성된 팔레트를 자동 추천합니다. 폰트 조합 추천폰트 선택컬러 팔레트의 분위기와 조화로우며, 가독성을 고려한 메인(Headline) 및 서브(Body) 폰트 조합을 제안합니다.직관적 시각화-추천된 조합을 배경색, 텍스트, 버튼 등에 즉시 적용하여 실제 디자인 예시로 보여줍니다.
🛠 기술 스택 : 이 프로젝트는 별도의 빌드 과정 없이 순수 웹 기술만을 사용하며, 빠른 프로토타이핑에 중점을 두었습니다.카테고리기술 스택FrontendHTML5, CSS3, JavaScript (Vanilla JS)Data PersistencelocalStorage (사용자가 확인한 조합을 임시 저장)
🚀 실행 방법 : 본 프로젝트는 정적인 웹 페이지로, 로컬 환경에서 웹 브라우저를 통해 바로 실행할 수 있습니다.파일 다운로드:git clone [저장소 URL]
cd Style-Matcher

애플리케이션 실행:프로젝트 폴더 내의 index.html 파일을 웹 브라우저(Chrome, Firefox 등)로 엽니다.사용 방법:화면에서 원하는 **'감성 키워드'**를 선택하면, 해당 키워드에 맞는 컬러 팔레트와 폰트 조합이 화면에 바로 시각적으로 표시됩니다.
